🎯 選擇排序法 (Selection Sort)

邏輯說明：

從待排序區塊中找出「最小值」，將它與該區塊的第一個數字交換，直到所有數字都排好為止。優點是交換次數比氣泡排序少。

JavaScript 實作範例：

function selectionSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        let minIdx = i;
        for (let j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIdx]) {
                minIdx = j; // 紀錄最小值的索引
            }
        }
        // 將最小值換到最前面
        [arr[i], arr[minIdx]] = [arr[minIdx], arr[i]];
    }
    return arr;
}

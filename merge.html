🧩 合併排序法 (Merge Sort)

邏輯說明：

採用「分而治之」法：不斷將數組對半拆解直到剩一個數字，再兩兩依照順序合併。時間複雜度穩定在 $O(n \log n)$，適合處理大數據。

JavaScript 遞迴實作：

async function mergeSort(arr, start, end) {
    if (start >= end) return;
    let mid = Math.floor((start + end) / 2);
    
    await mergeSort(arr, start, mid);    // 左半部
    await mergeSort(arr, mid + 1, end); // 右半部
    await merge(arr, start, mid, end);  // 合併
}

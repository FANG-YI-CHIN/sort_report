🔄 氣泡排序法 (Bubble Sort)

邏輯說明：

氣泡排序會比較相鄰的兩個元素，如果順序錯誤（前大後小）就交換。每一輪結束後，最大的數字會像「氣泡」一樣浮到最後面。

JavaScript 實作範例：

function bubbleSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交換數值
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
    return arr;
}

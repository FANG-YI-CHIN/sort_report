## 📈 複雜度分析摘要

| 演算法 | 時間複雜度 (平均) | 空間複雜度 | 穩定性 |
| :--- | :--- | :--- | :--- |
| Bubble Sort | $O(n^2)$ | $O(1)$ | 穩定 |
| Selection Sort | $O(n^2)$ | $O(1)$ | 不穩定 |
| Insertion Sort | $O(n^2)$ | $O(1)$ | 穩定 |
| Merge Sort | $O(n \log n)$ | $O(n)$ | 穩定 |
| Sedimentation Sort | $O(n + k)$ | $O(k)$ | 穩定 |

## 📝 實驗心得簡述
在本次電機系的演算法作業中，我透過 JavaScript 實作了視覺化界面。最大的收穫是理解到 **$O(n^2)$** 與 **$O(n \log n)$** 在數據量增大時的顯著差異。透過手寫「沈澱排序法」，我學會了如何利用空間權重來優化排序效率，並透過調整速度滑桿觀察遞迴與循環的運行。
